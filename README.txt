

					
					
		***************************STM32工程模板使用须知*************************************
		1.USER:放置工程，主函数等
		    (stm32f10xit.c：专门用来编写中断服务函数，stm32f10xconf.h配置外设驱动的函数库)
		2.FWlib：放置STM32内部模块（PWM、FTM、ADC）驱动。
		3.CMSIS：Cortex Microcontroller Software Interface Standard (标准协议层)
		    (core_cm3.c：为芯片外设提供一个进入CM3内核的接口,屏蔽不同编译器以及硬件的差异;
		     stm32f10x.h:包含STM32中配置寄存器地址和结构体类型定义;
		     system_stm32f10x.h：设置系统时钟和总线时钟)
		4.STARTCODE：放置芯片的启动文件
		5.Listing:放置连接生成的文件以及可执行文件等
	    *************************************************************************************
